### How about a *Plugin Architecture* for the bitmap effects?
- *Plugin architecture is a design pattern in software engineering where the application is structured in a way that allows pieces of its functionality, termed as ‘plugins’, to be added and removed seamlessly. These plugins are standalone components that interact with the main application, providing specific features or functionalities.*
  - https://dev.to/devleader/plugin-architecture-design-pattern-a-beginners-guide-to-modularity-4bo8
  - https://www.w3tutorials.net/blog/how-to-create-a-flexible-plug-in-architecture/
- Open up your final version of PE-7 Audio Visualizer
- Let's do the emboss code first, it's pretty easy to implement ...
- Create a new file named **src/image-effects.js**
- It looks like this:

```js
/**
 * Applies an emboss effect
 * @param {object} config - Configuration object containing data and parameters
 * @param {ImageData} config.imageData - https://developer.mozilla.org/en-US/docs/Web/API/ImageData
 * @param {object} config.params - The current drawing parameters from main.js
 */
export const emboss = ({imageData, params}) => {
  if(params.showEmboss){ 
    let data = imageData.data; 
    let length = data.length;
    let width = imageData.width;
    // note we are stepping through *each* sub-pixel
    for(let i = 0; i < length; i++){
        if(i%4 == 3) continue; // skip alpha channel
        data[i] = 127 + 2*data[i] - data[i+4] - data [i + width *4];
    }
  }
};
```

- Now import it at the top of **canvas.js** - `import * as imageEffects from "./image-effects.js";`
- Comment out the old emboss code in **main.js**
- For now, hard-code a call to the imported emboss code like this:

```js
const imageConfig = {
  imageData,
  params
};

imageEffects.emboss(imageConfig);
```

- Test the app now - emboss should still work

### Now we have 3 bitmap effects - (optinally) let's stop hard coding calls to them

- Add this to **canvas.js**
```js
const IMAGE_EFFECTS = [
  imageEffects.emboss,
  imageEffects.noise,
  imageEffects.invert,
];
```

- Now loop through the image effects and apply them!

```
for (let effect of IMAGE_EFFECTS){
    effect(imageConfig);
}
```
